# AI_Assistant
基于STM32F103开发AI助手

## 设备功能

- WIFI访问服务器ChatGPT
- 播放音乐，播放视频
- 液晶显示屏
- 外扩SD卡储存音频视频等文件(文件系统)
- 智能问答
- 屏幕UI
- 模拟USB
- 远程升级

## 上位机功能

- 音频转文字，文字转音频
- 本地ChatGPT部署
- UART通信
- WIFI/ETH通信

## 程序功能

- TIM6做定时器
- UART1做调试串口
- 2Kb I2C1 EEPROM
- 8MB SPI1 FLASH
- 三色呼吸灯
- SDIO SD卡
- SD卡支持FatFs
- FreeRTOS
- bootloader
- 模拟USB
